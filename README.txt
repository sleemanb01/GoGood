~~ This is a react native app

~ The App allow users to request help on current problems that require professionals,
And allow professional users to respond to them.


* It is developed with: React Native, TypeScript, .Net, SQL, Postman.

* The App is developed with the conseration of multiple code principles:
1- High cohesion.
2- Loose coupling.
3- Separation of concerns.
4- Reusablity.
5- Testability.
